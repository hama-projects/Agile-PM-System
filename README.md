# Agile PM System

## Overview

Agile PM System is a simplified project management platform built to explore Agile software development workflows and project tracking concepts.

The system provides core project management functionality such as sprint planning, task tracking, risk management, reporting, and predictive analytics in a single application.

This project was developed as a software engineering learning project focused on system design, analytics integration, and Agile methodologies.

---

## Features

### Project Management

* Sprint Management
* Kanban Board
* Task Tracking
* Risk Register
* Budget Tracking
* Release Planning
* Project Dashboard

### Analytics

* Productivity Scoring
* Velocity Forecasting
* Burnup Chart
* Sprint Timeline Analysis
* Product Roadmap Visualization
* Agile Maturity Scoring
* Sprint Success Prediction

### Predictive Features

* Task Delay Prediction
* Sprint Planning Assistance
* Bottleneck Analysis
* Risk Assessment Support
